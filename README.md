# AI Fire Robot Simulation

This project is an AI-enhanced firefighting robot simulation built using Python and Pygame.

The robot has two modes:

1. Manual Mode: The user controls the robot.
2. Automatic Mode: The robot detects fire, finds the nearest path, moves toward the fire, sprays water, refills the tank when needed, and continues searching for new fire zones.

## Features

- Manual and automatic robot movement
- Fire detection
- Pathfinding toward fire zones
- Water spraying system
- Tank refill system
- Visual simulation using Python/Pygame

## How to Run

Install requirements:

pip install pygame
than run the command
